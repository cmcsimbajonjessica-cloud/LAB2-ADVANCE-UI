---

Project Description

The Haircut Booking System is a simple web-based booking form that allows users to schedule haircut services online. Users can enter their name, select a date, choose a service, and submit a booking request.

---

Features

- Booking form for haircut appointment
- Service selection (Haircut, Shave, Hair Color)
- JavaScript alert for booking confirmation
- Simple responsive design using CSS
